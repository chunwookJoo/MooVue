- [x] 검색어를 입력해 영화를 검색할 수 있어야 합니다.
- [x] 검색된 결과를 통해 영화의 상세 정보를 볼 수 있어야 합니다.
- [x] 실제 서비스로 배포하고 접근 가능한 링크를 추가해야 합니다.
